How to Install

Desktop (Lightroom Classic)

  1. Open Lightroom Classic → Develop.
  2. Left panel → Presets → + → Import Presets.
  3. Select the .XMP files and import.

Mobile (Lightroom App)

  1. Download the .DNG files to your phone.
  2. Open Lightroom → Add Photos → import DNG.
  3. Open a DNG → tap ••• → Create Preset.

Tip: After applying a preset, fine‑tune exposure and white balance for best results.
